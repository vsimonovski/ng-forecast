# ng-forecast

## Technologies

* Angular@1.5.9, written in ES6
* Webpack

## Installation
* Clone or download repository
* Navigate to cloned/downloaded repository and run `npm i`
* Run `npm start`
* Application will be available on your localhost, port 8080

## Build
* Run `npm run build`
